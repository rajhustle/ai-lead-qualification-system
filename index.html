What this system does
1. Prospect chats via website, WhatsApp link, or email link
2. Structured 7-step BANT flow: name → company → need → budget → timeline → authority → contact
3. Scores across 4 dimensions: need fit (30pts), budget (25pts), urgency (25pts), authority (20pts) + contact bonus (10pts)
4. Hot = 70+ and gave contact. Warm = 40–69 and agreed to follow-up. Cold = declined OR low score
5. Rep brief auto-generated with score breakdown per dimension
6. CRM row copy includes column headers — paste directly into Google Sheet
7. In production: n8n automates CRM entry, WhatsApp, rep alert, stale lead follow-up
Scoring logic (prospect never sees this)
Need fit: Voice AI / lead qual = 30pts · Support/booking = 25pts · Other = 10pts
Budget: ₹1.5L+ = 25pts · ₹50k–1.5L = 18pts · Under ₹50k = 8pts · Undecided = 3pts
Urgency: Within 4 weeks = 25pts · This quarter = 18pts · Next quarter = 10pts · Exploring = 3pts
Authority: Decision maker = 20pts · Influencer = 14pts · Researcher = 7pts · Browsing = 2pts
Contact bonus: +10pts if valid phone + email and agreed to contact
Hot = 70+, Warm = 40–69, Cold = under 40 OR declined contact
Production n8n integration
Webhook trigger → Set node (format data) → Google Sheets (log all leads)
IF node routes by tier → Hot: WhatsApp now + Slack rep alert
Warm: WhatsApp + Wait 3 days + Gmail to rep
Cold: Newsletter add + Sheet tag
Cron 9am daily: check stale warm leads → auto WhatsApp follow-up
